Block vs. Inline Elements

This paragraph is made with the p element which is a block element.

This section is made with the div element which is also a block element. Sections of the webpage can be styled uniquely with div. Other block elements include h1-h6, ol, ul, table, li, etc.
This section uses div to style it differently from the other sections.

Inline elements only take up as much space as is necessary and do not start on a new line. For example, this is an inline element using span with a unique style. Other inline elements are a, abbr, cite, map, img, etc. While a block element can contain an inline element, an inline element cannot have a block element within it.
